# 人生の残り回数カレンダー

人生を「残り年数」ではなく「あと何回できるか」で眺める、静かなWebツールです。

桜を見られる回数。親と会える回数。
時間で見ると長い人生は、回数で見ると急に少なくなります。

## 特徴

- 入力は年齢から。すべての計算は端末の中だけで行われ、どこにも送信されません
- 目安の寿命・余命は、厚生労働省「令和6年簡易生命表」の平均余命をもとにした概算です
- 「週末を迎える回数」「温泉に行く回数」など、数えたい回数を自分で追加できます
- 結果は画像として保存できます

## 使い方

単一のHTMLファイルです。ブラウザで `index.html` を開くだけで動きます。
